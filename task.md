Задание: Создание форка репозитория и его клонирование в локальную среду
Цель:
Создать форк репозитория https://github.com/Ivvan20/my-fork-file.git, склонировать его на локальную машину и настроить связь с исходным репозиторием.

Шаги выполнения:
Создание форка на GitHub

Перейдите по ссылке исходного репозитория: https://github.com/Ivvan20/my-fork-file.git.

Нажмите кнопку Fork в правом верхнем углу страницы.

Выберите свой аккаунт GitHub в качестве места для форка.

Дождитесь создания копии репозитория в вашем аккаунте.

Клонирование форка на локальную машину

Откройте терминал (или Git Bash).

Выполните команду, заменив ваш-аккаунт на ваш логин GitHub:

bash
git clone https://github.com/ваш-аккаунт/my-fork-file.git
Перейдите в папку проекта:

bash
cd my-fork-file
Настройка связи с исходным репозиторием (upstream)

Добавьте исходный репозиторий как удаленный источник:

bash
git remote add upstream https://github.com/Ivvan20/my-fork-file.git
Проверьте список удаленных репозиториев:

bash
git remote -v
В выводе должны быть:

origin — ссылка на ваш форк.

upstream — ссылка на исходный репозиторий.

Проверка работоспособности

Убедитесь, что вы находитесь в ветке main (или master):

bash
git status
Выполните синхронизацию с исходным репозиторием (опционально):

bash
git fetch upstream
Итог:
Форк репозитория создан в вашем аккаунте GitHub.

Локальная копия склонирована и готова для работы.

Настроена связь с исходным репозиторием через upstream для получения обновлений.
